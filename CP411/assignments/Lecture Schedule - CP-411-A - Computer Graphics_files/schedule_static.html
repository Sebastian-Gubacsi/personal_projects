void Bresenham(int x1, int y1, int x2, int y2) {
    int Dx, Dy, x, y, s, p, c1, c2;
    if (x1 > x2) {
        x = x2;
        y = y2;
        x2 = x1;
        y2 = y1;
        x1 = x;
        y1 = y;
    } else {
        x = x1;
        y = y1;
    }

    Dx = x2 - x1;

    if (y1 > y2) {
        Dy = y1 - y2;
        s = -1;
        y2 = 2 * y1 - y2;

    } else {
        Dy = y2 - y1;
        s = 1;
    }

    setPixel(x, y);

    if (Dy <= Dx) {
        p = 2 * Dy - Dx;
        c1 = 2 * Dy;
        c2 = 2 * (Dy - Dx);

        while (x < x2) {

            x = x + 1;
            if (p < 0)
                p = p + c1;
            else {
                y = y + 1;
                p = p + c2;
            }
            if (s == 1)
                setPixel(x, y);
            else
                setPixel(x, y1 + y1 - y);
        }
    } else {
        p = 2 * Dx - Dy;
        c1 = 2 * Dx;
        c2 = 2 * (Dx - Dy);

        while (y < y2) {
            y = y + 1;
            if (p < 0)
                p = p + c1;
            else {
                x = x + 1;
                p = p + c2;
            }
            if (s == 1)
                setPixel(x, y);
            else
                setPixel(x, y1 + y1 - y);
        }
    }
}

Interactive graphics

As we know, mouse and keyboard are the common input devices to interact with a GUI program. The mechanism of interactions is that a mouse/keyboard action generates an event, which is captured by the program, then an event handler (also called event routine) is called to run.

Glut provides APIs to register event routines to input devices. We write event routines, functions, to define what to do when the event happens. Event can be mouse actions of left click, right click, and dragging, as well as key stroke of keyboard.

The programs in the following are C++ programs, it needs to use C++ compiler g++ to compile. If you want to use Eclipse to compile and run, drag it to A1’s MyOpenGL in Eclipse.

  • Right click to pop up a menu, and move mouse pointer to navigate the menu, and clicking left mouse key to select.
  • Read the program and understand how to create menu, register functions to mouse right click event Glut APIs.

  • Try different primitive names in drawPoints() function, e.g. GL_LINE_STRIP. Take a screen shot to verify that it works.

SimpleDraw (Programming)

Design and implement a C++ OpenGL program called SimpleDraw, an interactive 2D graphics program that allows users to draw rectangles and circles of different colors. 

  • A right click pop-up menu is used to select operations.
  • Shape objects can be drawed, re-positioned, re-ordered, and deleted.
  • The drawing can be saved as Scalable Vector Graphics (SVG) file. The saved SVG file can loaded into the program for editing or adding objects.
  • The drawing can also be exported to bitmap format as an artwork.

The detailed specifications are as follows.

Display window and menu

  1. Initial window properties: window size: 800x600, title: SimpleDraw (Your name), background color: white.
  2. Right clicking pops up a menu of the following menu operations or submenus.
  • New : It cleans up every thing and sets to the initial status.
  • Draw : It contains submenus of Rectangle and Circle. The Rectangle enables drawing rectangles. The Circle enables drawing circles.
  • Style : It contains submenus of Fill Color, Stroke Color, and `Stroke Width. The Fill Color has submenus of red, green, blue, black, white, and more colors of your choice. The Stroke Color has submenus of red, green, blue, black, white, and more colors of your choice. The Stroke Width has submenus of 1 to 5 values to specify the width of outline.
  • Edit : It has submenus of Select, Delete, Move front, Move back, Move around.
  • File : Contains submenus of Save SVG, Open SVG, and Export Bitmap.
  • Quit : It exits the program.

Data structures

  1. Design and implement a unified data structure to represent drawing objects, rectangles and circles with style attributes, fill color, stroke color and stroke-width of shape outline.
  2. Design and implement a double linked list to hold a list of objects, together with data structure operation functions to support the SimpleDraw operations.

Draw rectangle

  1. The program is initially set up for drawing rectangles in fill and stroke color red. User can change drawing style by the Style menu.
  2. The user presses and holds the left mouse button in the drawing area to determine a corner of the rectangle. The mouse is then moved to an opposite corner of the rectangle. Releasing the left mouse button draws the rectangle. When moving the mouse, an outline of a rectangle at the mouse position is showing (the robber band effect). Make sure that you are able to draw a rectangle in all directions from the initial point (i.e., not just to the bottom right corner).

Draw circle

  1. Similar to drawing rectangles, a user presses the left mouse button to determine the center of the circle. Holding the left button and moving the mouse, an outline of the circle passing the mouse position is showing. Releasing the left mouse button gets a point on the circle. User can change drawing style by the Style menu.
  2. It is required to use the Midpoint algorithm to draw circles.

Edit features

  1. Select: Use Edit -> Select to get into selection mode. Under this mode, when left mouse button is clicked, the program determines which object, if any, is underneath the mouse pointer. If there is more than one object under the mouse pointer, the program will select the top object. When an object is selected, it is high-lighted by a yellow outline. Deselect can be done by left clicking an empty point.
  2. Delete: When an object is selected, use Edit -> Delete to delete the selected object.
  3. Move front is to move the selected object to the top of objects. Move back is to move the selected object to the bottom of objects.
  4. Move around : click and hold left mouse on object to be moved, drag the object to a new position position, and release the mouse key.

Save/open SVG files

  1. Use menu File->Save SVG to save all the objects to SVG file of name: output.svg
  2. Use menu File->Open SVG to open the SVG file to open the output.svg file and import the object list.

Export to bitmap file

Use menu File->Save Bitmap to save the graphics image to bitmap file named output.bmp, which can be viewed by any bmp viewer, e.g., browser.

Circle & Square artwork

Use your SimpleDraw program to create an artwork consisting of circles and rectangles. Save your artwork in SVG and bitmap files, and rename them as c&s.svg, c&s.bmp.

Program design

Create project directory SimpleDraw under the al directory, and src directory under SimpleDraw. Put all source code files of the project under src. This can be done using Eclipse. For better management of the complexity, it is required to split the program into several files as follows.

  1. object.hpp and object.cpp is for the object data structure and double linked list data structures.
  2. edit.hpp and edit.cpp contains all functions and implementations for the edit operations.
  3. menu.hpp and menu.cpp contains menu functions.
  4. file.hpp and file.cpp contains file operation functions.
  5. simpledraw.cpp is the main function OpenGL program containing global variables, and mouse action functions.

The program can be compiled by command:

g++ -o simpledraw.exe edit.cpp file.cpp menu.cpp object.cpp simpledraw.cpp -lfreeglut -lopengl32 -lglu32

Alternatively, you can use makefile to compile and run the program. It should run like the simpledraw.exe this sample simpledraw program.

You can refer to this simpledraw reference design to design and implement your SimpleDraw program.
